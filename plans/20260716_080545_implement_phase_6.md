# Plan - Phase 6: Splash Screen & About Page

**Status:** completed

This plan implements Phase 6 of the Devi (Navadurga) project. It introduces a blooming splash screen on launch with custom vector drawing and sound, and a verified metadata-driven About screen linked from the Carousel screen.

---

## 1. What the issue is

Currently, the app starts directly on the `CarouselScreen`. The transition from launch needs to feel sacred, immersive, and visually premium:
- **Splash Screen**: A closed lotus bud blooming into a full Rakta Kamala with a golden aura over 1.5 - 2 seconds, playing the temple bell chime sound (`chime.mp3`) simultaneously.
- **About Screen**: An info button on the AppBar that currently shows a simple SnackBar needs to navigate to a beautiful, conforming About screen. This screen must show metadata loaded from `app_config.json` via `ConfigService` (validating version and build matches using `package_info_plus`).

---

## 2. Plan for the fix

1. **Create Splash Screen (`lib/screens/splash_screen.dart`)**:
   - Implement `SplashScreen` as a `ConsumerStatefulWidget`.
   - Build a custom `LotusPainter` that uses `CustomPainter` to draw a realistic blooming lotus. The petals will dynamically tilt, widen, and spread based on `bloomProgress` (0.0 to 1.0). Fill petals with Rakta Red (`#8B0000`) and stroke with Swarna Gold (`#D4AF37`).
   - Run a 2-second animation combining scale (0.3 to 1.0), rotation (0 to 1.5 rotations), and bloom progress (0.0 to 1.0).
   - Play the chime sound effect (`assets/audio/sfx/chime.mp3`) via `AudioService` immediately on launch.
   - Pre-warm the `appConfigProvider` and `durgasProvider` in the background during the animation.
   - When the animation completes and data loading finishes, navigate to `CarouselScreen` using a smooth `PageRouteBuilder` fade transition.

2. **Create About Screen (`lib/screens/about_screen.dart`)**:
   - Create `AboutScreen` displaying config parameters: `appName`, `description`, `version`, `build`, and custom details (Author, License, AI used, IDE used).
   - Use `package_info_plus` to verify the actual build version/number matches the config.
   - Conform to the devotional theme: Haridra background, Swarna Gold bordered card, and Rakta Red titles/icons.

3. **Update App Entrance (`lib/app.dart`)**:
   - Set the initial route/home screen to `SplashScreen` instead of `CarouselScreen`.

4. **Update AppBar Action in Carousel Screen (`lib/screens/carousel_screen.dart`)**:
   - Change the `IconButton` action to navigate to `AboutScreen`.

5. **Create Tests**:
   - Write tests in `test/splash_screen_test.dart` and `test/about_screen_test.dart` to verify the splash animation triggers, the about screen loads and displays the config values.
